# Berends Arbo & Veiligheid BV

Static website project for Berends Arbo & Veiligheid BV.

## Project structure

```
berends-arbo-veiligheid/
├── index.html
├── diensten.html
├── projecten.html
├── over-mij.html
├── contact.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── assets/
│   ├── images/
│   └── videos/
├── README.md
└── .gitignore
```

## Setup

No build tools or dependencies required. Open any `.html` file directly in a browser, or use a local development server such as [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for VS Code / Cursor.
